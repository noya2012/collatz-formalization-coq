# Unstable / Work in Progress

This directory contains **experimental and in-development** Coq proof files for the Collatz conjecture formalization project.

## Contents

- `collatz_part_20_l.v` — Left part of part 20 (fixed point equations for canonical macrostep)
- `collatz_part_20_r.v` — Right part of part 20 (list operations and utilities)
- `hard.v` — Auxiliary lemmas and helper proofs

## Status

**These files are actively being developed and are not yet stable.**

The proofs here may be incomplete, refactored, or have known issues. They are subject to change without notice.

## Usage

These files depend on the main Coq development. Do not rely on them for production or published results until they are moved to the main project directory.
